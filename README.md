# 沉浸式穿越加载器

> 一个 Markdown 文件 + LLM = 多世界沉浸式角色扮演

## 如何开始

1. 复制 `game.md` 全部内容
2. 粘贴到 DeepSeek、豆包、GPT 等 AI 对话框
3. 输入"开始游戏"

AI 会引导你选择世界、创建角色、随机天气，然后你就可以自由探索了。

## 可选世界

| 世界 | 文件 | 时代 | 特色 |
|------|------|------|------|
| 清明上河图 | `word/qingmingshanghe.md` | 北宋汴京 | 历史沉浸、民俗风情、盛世余晖 |
| 武林外传 | `word/wulinwaizhuan.md` | 明代七侠镇 | 武侠喜剧、江湖恩怨、市井江湖 |

> 如需更完整体验，可同时发送 `game.md` + 对应世界文件给 AI。

## 核心特性

- 多世界切换——一个规则引擎加载不同世界
- 动态天气——开局随机，4-6回合可能变天，影响NPC/物价/场景
- 流动时间——06:00-24:00自然流逝，NPC按时段作息
- 防漂移——快照唯一真相源+强制自检+规则检查清单+上下文刷新
- 纯Markdown驱动——无需代码引擎，复制即玩
