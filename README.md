# 🎮 沉浸式穿越加载器

> **一个 Markdown 文件 + LLM = 多世界沉浸式角色扮演体验**

## 📖 项目简介

本项目是一个基于 Markdown 和 LLM 的**沉浸式文本冒险系统**。
不同于传统的文字冒险游戏，本系统没有固定的选项和剧本。你是一个**穿越者**，可以自由选择进入不同的世界：

- 🏮 **北宋汴京** —— 走进《清明上河图》，体验千年前的市井繁华
- ⚔️ **七侠镇** —— 踏入《武林外传》的同福客栈，开启你的江湖人生

时间自然流逝，天气随机变幻，NPC 自主生活，而你将书写属于自己的穿越故事。

---

## ✨ 核心特性

- **🌍 多世界切换**：一个规则引擎，支持多个世界（清明上河图、武林外传，可扩展）
- **🌤️ 动态天气系统**：开局随机天气，每4-6回合可能变天，天气影响一切——衣着、出行、物价、NPC行为
- **🌏 薛定谔的世界**：你未到达之处皆为虚空，你到达之时 AI 根据时间与规则即时生成
- **⏳ 流动的时间**：游戏时间自然流逝，NPC 会根据时段作息，事件会随时间发酵传播
- **🧑 丰富的 NPC 数据库**：每个世界都有数十到数百个独立角色，有姓名、口音、职业和生活轨迹
- **📜 严格的世界还原**：基于史料或原作，严格遵循服饰、饮食、建筑与礼仪
- **🔒 上下文防丢失**：YAML 元数据追踪世界状态，每次回复强制声明当前世界和天气
- **📝 纯 Markdown 驱动**：无需任何代码引擎，复制文件给 AI 即可开始游玩

## 🚀 如何开始游玩

### 方法一：完整体验（推荐）

1. **复制文件**：将 `game.md` 和对应的世界文件发送给 AI：
   - 想玩**清明上河图** → 同时发送 `word.md`
   - 想玩**武林外传** → 同时发送 `wulinwaizhuan.md`
2. **输入指令**：发送 **"开始游戏"** 或 `/start`
3. **选择世界**：AI 会询问你想进入哪个世界
4. **创建角色**：回答 AI 提出的身份设定问题
5. **天气随机**：AI 掷骰确定开局天气，用沉浸式文字呈现
6. **自由探索**：用自然语言与 AI 交互

### 方法二：单世界快速开始

直接发送对应的世界文件 + "开始游戏"：
- 清明上河图：`word.md` + "开始游戏"
- 武林外传：`wulinwaizhuan.md` + "开始游戏"

### 交互示例

```
玩家：我走进城门，看看有什么新鲜事。
玩家：我跟那个剃头匠搭话。
玩家：下雨了，我去茶馆避避雨。
玩家：我要使出排山倒海！
玩家：佟掌柜，这雨什么时候能停啊？
```

## 🌤️ 天气系统

**天气影响一切**——衣着、出行、心情、物价、NPC 行为。

### 天气类型

| 天气 | 氛围 | 清明上河图 | 武林外传 |
|------|------|-----------|---------|
| ☀️ 晴朗 | 明媚/万里无云 | 春日暖阳照汴河 | 关中大晴天 |
| ⛅ 多云 | 阴晴不定 | 汴河倒影朦胧 | 屋顶看云 |
| 🌧️ 小雨 | 烟雨蒙蒙 | 油纸伞/青石板 | 廊下听雨 |
| 🌧️ 大雨 | 倾盆如注 | 避雨茶馆 | 客栈躲雨 |
| 💨 大风 | 狂风呼啸 | 风沙漫天 | 关中北风 |
| 🌫️ 浓雾 | 雾锁城池 | 晨雾中汴河 | 七侠镇雾锁 |

### 天气规则

- **开局随机**：掷骰确定初始天气
- **动态变化**：每4-6回合30%概率变天
- **真实影响**：
  - 晴天 → 市集热闹，NPC 外出，物价正常
  - 雨天 → 行人稀少，茶馆生意好，伞具涨价
  - 大风 → 漕船停航 / 轻功受影响
  - 浓雾 → 能见度低，神秘事件触发率提高
- **NPC 反应**：NPC 会根据天气调整行为（雨天不出摊、天冷加衣）

## ⚙️ 系统设计理念

### 1. 规则与数据分离
我们将**规则逻辑**（`game.md`）与**世界内容**（`word.md`, `wulinwaizhuan.md`）彻底分离。这意味着你可以：
- 随时切换不同的世界体验
- 轻松创建新的世界（唐代长安、medieval 欧洲、赛博朋克 2077）
- 规则文件保持不变，降低扩展成本

### 2. 强制上下文追踪
通过 YAML 元数据追踪关键状态，确保 AI 不会：
- 遗忘当前处于哪个世界
- 混淆不同世界的设定
- 忽略当前天气和时间
- 丢失上下文

每次回复前，AI 都会声明：`[世界：xxx | 天气：xxx | 时间：xxx | 回合：xxx]`

### 3. 天气驱动叙事
天气不只是装饰，它驱动整个世界的运转：
- 雨天 → 茶馆里的人多了，故事就多了
- 大雾 → 看不清路，意外相遇的概率高了
- 晴天 → 集市热闹，商机来了
- 每次叙事都融入天气，让世界"活"起来

### 4. AI 的角色
AI 不仅仅是讲故事的人，它是：
- **DM（地下城主）**：引导探索，不替玩家做决定
- **世界引擎**：维持时间流逝、天气变化、事件因果链
- **所有 NPC**：通过严格的 NPC 数据库，确保每个人物的言行符合其身份和籍贯
- **天气预报员**：根据天气调整整个世界的氛围和行为
