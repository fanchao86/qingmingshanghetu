# 🏮 清明上河图·沉浸式穿越系统

> **一个 Markdown 文件 + LLM = 沉浸式古代生活模拟体验**

## 📖 项目简介

本项目是一个基于 Markdown 和 LLM 的**沉浸式文本冒险系统**。
不同于传统的文字冒险游戏，本系统没有固定的选项和剧本。你是一个**穿越者**，进入了一个由 LLM 实时推演的**活着的北宋汴京**。

在这里，你可以自由探索、与数百个 NPC 对话、参与市井事件。
时间在流逝，世界在运转，而你，将书写属于自己的汴京故事。

---

## ✨ 核心特性

- **🌏 薛定谔的世界**：你未到达之处皆为虚空，你到达之时 AI 根据时间与规则即时生成。
- **⏳ 流动的时间**：游戏时间自然流逝，NPC 会根据时段作息，早上的事晚上会传成流言。
- **🧑 280+ NPC 数据库**：每个人有独立的名字、口音、职业和生活轨迹。
- **📜 严格的历史还原**：基于《清明上河图》与北宋史料，严格遵循服饰、饮食、建筑与礼仪。
- **📝 纯 Markdown 驱动**：无需任何代码引擎，复制文件给 AI 即可开始游玩。

---

## 🚀 如何开始游玩

1. **复制文件**：将 `game.md`, `word.md`, `npc.md` 发送给任意支持长文本的 AI（如 ChatGPT, Claude, Gemini 等）。
2. **输入指令**：在对话框输入 **"开始游戏"** 或直接发送 `game.md` 的内容。
3. **创建角色**：回答 AI 提出的身份设定问题（如名字、身份）。
4. **自由探索**：用自然语言与 AI 交互，例如：
   - *"我走进城门，看看有什么新鲜事。"*
   - *"我跟那个剃头匠搭话。"*
   - *"我等到下午再去茶馆。"*

## 📂 文件结构

```text
game-qingming/
├── game.md       # 🎮 系统规则：AI 行为指令、时间机制、回复格式（通用规则层）
├── word.md       # 🌍 世界设定：北宋背景、33 个场景、56 个事件、文化约束（数据层）
├── npc.md        # 🧑 NPC 数据库：280+ 角色详细数据（NPC 库）
└── README.md     # 📖 项目说明
```

### 文件职责说明

| 文件 | 职责 | 说明 |
| :--- | :--- | :--- |
| **`game.md`** | **规则引擎** | 定义了 AI 如何扮演 DM、时间如何流逝、如何响应玩家指令。此文件通用，可用于加载不同世界。 |
| **`word.md`** | **世界设定** | 包含了《清明上河图》的世界观、33 个地标场景、56 个历史事件库及严格的文化约束。 |
| **`npc.md`** | **角色库** | 详细记录了 280 多个 NPC 的姓名、职业、籍贯、口音、穿着和位置。 |

## ⚙️ 系统设计理念

### 1. 规则与数据分离
我们将**规则逻辑**（`game.md`）与**世界内容**（`word.md`, `npc.md`）彻底分离。这意味着你可以随时替换 `word.md` 和 `npc.md` 来创造全新的世界（如：唐代长安、中世纪欧洲、赛博朋克 2077），而规则文件保持不变。

### 2. AI 的角色
AI 不仅仅是讲故事的人，它是：
- **DM（地下城主）**：引导探索，不替玩家做决定。
- **世界引擎**：维持时间流逝、事件因果链。
- **所有 NPC**：通过严格的 `npc.md` 数据库，确保每个人物的言行符合其身份和籍贯。
